1. DYNAMODB TABLE SETUP:
   - Go to DynamoDB Console.
   - Create a table named: user-state
   - Partition Key: userId (String)
   - Capacity: On-Demand (to save money)

2. IAM PERMISSIONS (CRITICAL):
   - Go to your LF1 Lambda function in the console.
   - Click 'Configuration' -> 'Permissions' -> Click the Role Name.
   - Add an Inline Policy to the role allowing DynamoDB actions:
     {
       "Version": "2012-10-17",
       "Statement": [
         {
           "Effect": "Allow",
           "Action": [
             "dynamodb:PutItem",
             "dynamodb:GetItem"
           ],
           "Resource": "arn:aws:dynamodb:REGION:ACCOUNT_ID:table/user-state"
         }
       ]
     }
